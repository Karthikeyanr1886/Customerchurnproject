# Customer Churn Prediction

This project predicts whether a customer will leave (churn) using machine learning techniques.

## Overview

The dataset is processed using Pandas and NumPy. Data visualization is performed using Matplotlib and Seaborn to analyze customer behavior and trends.

## Models Used

* Logistic Regression
* Decision Tree
* Random Forest

## Features

* Data cleaning and preprocessing
* Handling missing values
* Exploratory data analysis
* Data visualization
* Model training and evaluation
* Hyperparameter tuning using GridSearchCV

## Technologies

* Python
* Pandas, NumPy
* Matplotlib, Seaborn
* Scikit-learn

## Goal

To identify customers who are likely to churn so that businesses can take preventive measures.

## How to Run

1. Install required libraries:

   ```
   pip install numpy pandas matplotlib seaborn scikit-learn
   ```
2. Run the script:

   ```
   python main.py
   ```
